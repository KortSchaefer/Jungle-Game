JUNGLE GAME SCALING + PROGRESSION REFERENCE
Source: renderer code (schemaVersion 6)

1) CORE ECONOMY FORMULAS

- Tree cost: treeBaseCost * (treeCostGrowth ^ treesOwned)
  - treeBaseCost = 25
  - treeCostGrowth = 1.12

- Worker cost: workerBaseCost * (workerCostGrowth ^ workersOwned)
  - workerBaseCost = 40
  - workerCostGrowth = 1.15

- Building cost: baseCost * (growth ^ level)
  - Packing Shed: base 750, growth 1.35
  - Fertilizer Lab: base 1200, growth 1.35
  - Research Hut: base 1800, growth 1.35

- Orchard cost: orchardBaseCost * (orchardCostGrowth ^ orchardsOwned)
  - orchardBaseCost = 4000
  - orchardCostGrowth = 1.22

- Orchard pick rate: orchardsOwned * orchardPickRatePerSecondPerOrchard
  - orchardPickRatePerSecondPerOrchard = 0.85 picks/sec

- Market price per banana:
  - base = 0.35
  - multiplied by tree tier value multiplier:
    T1 1.00, T2 1.15, T3 1.35, T4 1.65, T5 2.05, T6 2.60, T7 3.40

- Auto-sell fallback price: 0.20 per banana

- Buyer effective price:
  marketPrice * buyerMultiplier * exportPriceMultiplier * antimatterBoost * repMultiplier * laneMultiplier * eventMultiplier
  - repMultiplier = 1 + (reputation * 0.004)
  - eventMultiplier: 1.10 during Squirrel Market Surge, else 1

- Shipment validation:
  shipmentAmount must be between:
  - buyer.minShipment
  - min(buyer.maxShipment, selectedLane.capacity)

- Buyer cooldown after shipment:
  buyer.cooldownSeconds * exportCooldownMultiplier
  (minimum 1 second)

2) PRODUCTION + HARVEST SYSTEM

Default tree-harvest state:
- maxBananasOnTree = 12
- spawnInterval = 1.5 sec
- clickHarvestYield = 1
- goldenChance = 0.005 (0.5%)
- goldenMultiplier = 35x
- diamondChance = 0.0005 (0.05%)
- diamondMultiplier = 200x
- shakeCooldownSeconds = 35

Spawn region:
- X in middle third of canopy (33.33% to 66.67%)
- Y in top 2/5 of canopy (0% to 40%)

Auto-picker pacing caps:
- Worker picks capped to 80% of spawn-rate target
- Max worker picks per update = 2
- Max orchard picks per update = 3
- Max monkey picks per update = 2
- Bananas younger than 450 ms are protected from auto-picks

Derived harvest modifiers from gameplay:
- Trees owned adds canopy capacity: +floor(treesOwned * 0.75)
- Trees owned speeds spawn with diminishing returns:
  spawn multiplier = 1 / (1 + sqrt(treesOwned) * 0.14)
- Fertilizer Lab:
  - spawn speed multiplier factor: 1 / (1 + level * 0.06)
  - canopy bonus: +floor(level * 0.5)

3) BUILDING EFFECTS

- Packing Shed:
  export price multiplier = 1 + (level * 0.04)

- Fertilizer Lab:
  production support via harvest system (spawn/capacity improvements)

- Research Hut:
  upgrade cash discount multiplier = max(0.55, 1 - level * 0.03)

- Research Points gain:
  RP/sec = researchHutLevel * 0.06 * achievementResearchMultiplier

11) CONTRACTS + REPUTATION

- Max active contracts: 3
- Contract generation interval: every 45s (game checks loop)
- Premium contracts unlock at buyer reputation >= 60
- Successful shipment to buyer: +0.9 reputation
- Failed shipment attempt (not enough bananas): -0.8 reputation

Contract generation details:
- targetBananas = floor(buyer.minShipment * scale + random * buyer.maxShipment * 0.4)
  - scale: 2.1 premium, 1.35 standard
- duration: 10 min premium, 6 min standard
- reward cash: targetBananas * effectiveBuyerPrice * (1.4 premium or 0.95 standard)
- reward rep: +8 premium, +4 standard

13) PRESTIGE (PIP)

Unlock condition:
- treeTierIndex >= 5 (Quantum Banana Reactor)
  OR
- totalBananasEarned >= 1,000,000

PIP gain formula:
- floor((log10(totalBananasEarned + 1) - 5) * 5), min 0

Permanent bonuses from PIP:
- production multiplier: 1 + pip * 0.02
- export price multiplier: 1 + pip * 0.015
- click multiplier: 1 + pip * 0.01
- worker prestige bonus also contributes in worker stat: 1 + pip * 0.005

18) IMPLEMENTED TUNING UPDATE (POST-SUGGESTED-SCALING)

The engine now applies these additional scaling rules:

- CEO/global multiplier:
  ceoMult = 1.03 ^ (ceoLevel - 1)
  where ceoLevel = floor(log10(totalBananasEarned + 1)) + 1
  Applied to: production multiplier and export price multiplier.

- Tier value multipliers for market price:
  [1, 1.25, 1.6, 2.2, 3.4, 5.5, 9.0]

- Reputation multiplier (superlinear):
  repMult = 1 + 0.0025*rep + 0.00003*(rep^2)

- Shipping lane capacity scaling by tier:
  effectiveCapacity = floor(baseCapacity * (1 + treeTierIndex * 0.05))
  Shipment cap remains:
  min(buyer.maxShipment, effectiveLaneCapacity)

- Trees -> spawn interval scaling updated:
  spawnMultFromTrees = 1 / (1 + log1p(treesOwned) * 0.35)

- Picker cap scaling by tier (per update):
  worker cap = 2 + floor(treeTierIndex / 2)
  orchard cap = 3 + floor(treeTierIndex / 2)
  monkey cap = 2 + floor(treeTierIndex / 3)

- Packing Shed tier synergy:
  tierIndex >= 3: packing multiplier x1.25
  tierIndex >= 5: packing multiplier x1.6

- Prestige (PIP) bonuses now slightly superlinear:
  production = 1 + 0.02*pip + 0.0002*(pip^2)
  export     = 1 + 0.015*pip + 0.00015*(pip^2)
  click      = 1 + 0.01*pip + 0.0001*(pip^2)

- Research row/category completion multipliers:
  Farming Tech:
  row complete: production x1.12 each
  full category complete: production x1.25

  Logistics:
  row complete: export x1.12 each
  full category complete: export x1.25

  Finance:
  row complete: export x1.10 each, click x1.05 each
  full category complete: export x1.20

  Weird Science:
  row complete: click x1.14 each, production x1.06 each
  full category complete: click x1.25
